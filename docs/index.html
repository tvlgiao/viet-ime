Tính năng

Mọi thứ bạn cần để gõ tiếng Việt trên Windows, macOS & Linux.

🌐

Hoạt động System-wide

Gõ tiếng Việt trên mọi ứng dụng — Office, trình duyệt, Notepad, và tất cả app trên Windows, macOS và Linux.

⌨️

Hỗ trợ Terminal & IDE

Hoạt động hoàn hảo trên Terminal, PowerShell, Warp, VS Code Terminal — nơi Unikey/EVKey thường gặp lỗi.

🔤

Telex & VNI

Hỗ trợ đầy đủ 2 kiểu gõ phổ biến nhất. Chuyển đổi nhanh qua system tray hoặc cửa sổ cài đặt.

🧠

Nhận diện từ tiếng Anh

Tự động phát hiện từ không phải tiếng Việt (windows, download...) và bỏ qua xử lý dấu — giảm lỗi khi gõ lẫn.

⚡

Portable — Không cần cài

Windows: tải 1 file EXE chạy ngay. macOS: mở DMG, kéo vào Applications. Linux: chạy AppImage hoặc binary.

🔄

Dùng song song IME khác

Windows: tự tắt khi có Unikey. macOS/Linux: Ctrl+Shift tắt nhanh khi dùng bộ gõ khác. Không xung đột.

🎨

System Tray & Dark UI

Icon tray thay đổi theo trạng thái bật/tắt. Menu context dark theme đẹp mắt. Cửa sổ cài đặt gọn gàng.

📐

Đặt dấu đúng chuẩn

Xử lý đúng quy tắc đặt dấu tiếng Việt: ươ, oa, oe, qu, gi — kể cả các trường hợp phức tạp.

🛡️

An toàn & Mã nguồn mở

Không thu thập dữ liệu, không kết nối internet. Mã nguồn mở trên GitHub, MIT License.

Cài đặt

Chọn cách cài phù hợp với bạn.

Cài bằng 1 lệnh

$ curl -sSL https://raw.githubusercontent.com/donamvn/viet-ime/master/scripts/install.sh | bash
PS> irm https://raw.githubusercontent.com/donamvn/viet-ime/master/scripts/install.ps1 | iex

Tự tải, cài vào PATH, tạo shortcut. Gỡ: thêm --uninstall

── hoặc tải thủ công ──
1

Tải xuống

Windows: VietIME.exe
macOS: VietIME.dmg
Linux: VietIME.AppImage

2

Cài đặt nhanh

Windows: chạy trực tiếp EXE.
macOS: mở DMG, kéo vào Applications.
Linux: chmod +x rồi chạy.

3
